# LuaSTG Retro update changelog

* LuaSTG-Retro v0.0.2 (pre-release)
    * API
        * API change: Resource pools are now created and managed through `lstg.ResourceManager`.
        * API change: Resource loading and creation APIs are now methods of `lstg.ResourcePool`.

* LuaSTG-Retro v0.0.1 (pre-release)
    * API
        * API addition: `lstg.SwapChain.ScalingMode.integer_aspect_ratio ` has been added.
        * API addition: `lstg.DiscordRPC` has been added.
    * Archive
        * Support for managing game resources and assets using the DAT archive format is now available.
    * Misc
        * Translation of the entire console
